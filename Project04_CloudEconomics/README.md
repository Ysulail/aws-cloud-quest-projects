# Cloud Economics

## 📌 Project Overview
This project involves estimating AWS costs using the AWS Pricing Calculator.

## 🚀 Key Features & Services
- **Amazon EC2** for computing.
- **EBS Storage** for persistent storage.
- **Data Transfer** cost calculations.
